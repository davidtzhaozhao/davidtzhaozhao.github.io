穆斯林人口暴增将吞噬世界?

三月 31, 2017 全球 本文总阅读量次

美国新任总统特朗普宣布禁止部分伊斯兰国家人民入境，此事引起了美国国内自由派人士及记者们的反弹，有法官甚至对抗到底，拒绝执行。

但是国内民众却又有56% 的人表示支持，为什么？这个事件所代表的背后涵意是什么？这是本周我们将讨论的主题。

　　如果你把特朗普的这项政策看成这是美国对移民人口的控制，那您恐怕就大错特错，表面是如此，但骨子里却是一场“基督文明”和“穆斯林文明”的战争。

　　这件事还得从2011年皮尤研究中心的一篇研究报告说起。根据研究报告 （翻译内容节录自新浪微博）

　　一个文化若要能持续超过25年，那么人口的出生率必须达到每个家庭有2.11个孩子 ，否则的话，这个文化将会渐渐衰落。从历史上看，仍未见有一个文化 ，在人口出生率低于 1.9 的情况下能出现转机的。

　　而 1.3 的出生率更是不可能逆转 ，因为一旦走入这个地步，它将要花80到100年的时间来自我修正 。可是因为没有任何经济模式 ，能在这期间持续支撑着这个文化变成不可能 。换句话说 ，若果有两对父母，每对有一个孩子 。那每个父母就只有1/2个孩子 ，如果那些孩子再生一个孩子 ，那每个祖父母就只能1/4个孙子了 。如果2006年只有1百万个婴儿出生 ，到了2026年，就很难有2百万个成年人和劳动力 。随着人口萎缩 ，文化也随之而萎缩 。

　　在2007 年，法国的人口出生率为1.8 、英国：1.6 、希腊：1.3、德国：1.3 、意大利：1.2 、西班牙：1.1。 整个欧盟的31个国家 ，人口出生率仅为1.38 。

　　历史研究告诉我们： 这些数据表示欧洲的文化不能逆转 。若干年以后，我们认识的欧洲将不复存在 。

　　但欧洲的人口至今却没有下降，为什么？ 是移民 ，来自伊斯兰国家的移民 ！

　　自1980年以来，在欧洲的人口增长中 ，有90%是伊斯兰国家的移民 。在法国，每个家庭有1.8个孩子 ，穆斯林家庭却有8.1个孩子。在法国南部 ，过往是教堂最多的地区之一，世界闻名 ，但现在那里的清真寺比教堂还要多 。而20岁或以下的儿童有30%是穆斯林，在一些较大的城市如尼斯、马赛和巴黎，数字更达到45% 。到了2027年 ，5个法国人中就有一个穆斯林，只需39年时间 ，法国将变成一个伊斯兰共和国 。

　　在过去30年中，英国的穆斯林人口 ，从8.2万增加到250万 ，增加了30倍 。英国现在有1000多座清真寺 ，而许多前身更是教堂 。

　　在荷兰，50%的新生婴儿是穆斯林，只要15年时间 ，荷兰有一半人口将是穆斯林。

　　在俄罗斯，有超过2300万的穆斯林。即每5个俄罗斯人就有一个是穆斯林 ，不出几年 ，俄罗斯军队里就有40%的人员是穆斯林。

　　在比利时，目前有25%的人口和50%的新生婴儿是穆斯林。比利时政府称，到了2025年 ，仅17年之后 ，比利时 1/3 的儿童将出生在穆斯林家庭 。

　　德国是第一个公开谈到这个问题的国家 。它的政府最近公布一份报告说 ，“(德国)人口下降趋势已无法停下来 ，而且越来越严重已经不能逆转 ，到了2050年 ，她将变成一个穆斯林国家”。

　　(利比亚总统)卡达菲说： “有迹象显示 ，安拉要使伊斯兰在欧洲大获全胜 。不用动刀枪，不用攻城略地 ，我们不需要恐怖分子，我们不需要自杀式的炸弹攻击 ，我们在欧洲的5千多万穆斯林 ，只需几十年就能将它转变为穆斯林大陆” 。

　　在北美 ，各种数字说出类似的情况 。现在加拿大的人口出生率是1.6 ，比起能延续文化所需的出生率低了4个点 。而伊斯兰现在是增长最快的。

　　在美国，目前的人口出生率是1.6 。算进拉丁美洲的移民 ，出生率增至2.11 ，是维持文化所需的最低值 。在1970年，美国有10万穆斯林。今天，就超过了900万。

　　2009年全球23%人口是穆斯林，1990至2012年，世界穆斯林人就每年以2.2%的速度增长着，至2023年穆斯林人口占全球人口比例将剧升至26.4%，第一次超过基督人口（含天主教、新教及正教）。2050年，穆斯林人口将占全球人口超过三分之一。

　　我们再追溯得远些，1950年，全球人口25亿，穆斯林仅有2亿，占全球人口8%，若2050年增加到三分之一，那等于是4倍，依这个趋势下去，人类终有一天会成为“穆斯林的世界”。所以美国紧张了。

　　澳洲还好，在1970年代之前，澳洲几乎是没有穆斯林的，但六日战争、两伊战争、阿富汗战争、伊拉克战争…，之后，澳洲为了履行国际义务，开始接收战乱中的难民，加上后来大批“船民”涌入，但是到目前为止，澳洲穆斯林人口仅占约占总人口的2.3%（含中东及东南亚穆斯林人），比印度教、佛教还低，（感谢当年澳洲总理霍华德英明，把船民挡在境外，才免除了澳洲一场浩劫）。
　　
　　所以虽然澳洲穆斯林新生人口比率仍然比其他族裔来得高，但是要成为多数还有很长的路要走。既使如此，从报纸上看到的有关澳洲伊斯兰青年参加恐怖组织，马丁广场爆炸案…所引起的恐慌，已足以令澳洲政府头疼了。

　　中国有多少穆斯林？官方的统计是2,300万人（和台湾总人口差不多），但民间统计是这个数字的一倍，约5,000万，西方传媒更夸张说中国穆斯林达1.2亿，由于中国人口普查不含“宗教”这个选项，所以很难有一个正确的数字，人们祗能依少数民族来分类，如：回族、维吾尔族、哈萨克族、东乡族、柯尔克孜族、撒拉族、塔吉克族、乌孜别克族、保安族、塔塔尔族，都是穆斯林来推测穆斯林人口，但汉人、满人、蒙古人也有穆斯林的，所以很难做正确统计中国穆斯林人口。
