从华为“闪存门”看国产手机之殇：被抓住的命门

六月 9, 2017 科技 本文总阅读量次

如今放眼四海，如果要选一家令国人最骄傲的民族公司，我想没有多少人会犹豫把这个头衔送给这样一家技术型的公司：华为。

被黑的体无完肤的华为

很多人觉得为什么不是阿里、腾讯呢？它们不也很伟大，但是我们要知道，中国没有阿里和腾讯，其他互联网公司可能取而代之，而且它们能发展壮大是极大的受益于中国的GFW，然而华为作为一家披荆斩棘面向全球的公司，发展到这样的地步，没有任何一家公司能够替代！

然而，就是这样一家令国人骄傲的公司最近却陷入了大麻烦，被质疑“偷工减料”，批的体无完肤，搞不好陷入品牌危机，这就是华为P10的“闪存门”事件了：因为被网友质疑在最新的旗舰P10上使用了UFS2.0、UFS2.1和emmc5.1三种闪存，且没有告知消费者，都按照统一的价格卖，华为被网友黑得体无完肤，以致于华为终端和余承东先后出来发表澄清新闻。

坦白说，这声明没有丝毫诚意，这事华为做的太不厚道。P10手机上用2种不同性能的闪存，理论上相差两倍多的速度。虽然华为说自己能做到软硬件优化，不影响用户体验，但不管怎样，同一价格手机两种不同性能闪存对消费者而言，有何公平可言，这样岂不是买手机，好坏还得靠运气？

当然，华为给出的理由，也是既心酸又无奈：缺货、缺货、缺货！

2016年，国产手机的天堂

华为是当之无愧国产手机的骄傲，也着实改变了以往国产手机的形象。12年前，国产手机=山寨手机。什么金鹏、汉泰、CECT、中天、德赛、东极星……你连听都没听过。

那个时候，我们的确是个“山寨”国度，可是山寨又怎么了，那个时候我们培养了一批手机零件供应商，我们卧薪尝胆，使得中国形成了强大的手机供应链，并培养了大量的工艺，模具，PCB，屏幕，结构等等人才。

后来的事，很多人都清楚，随着14、15年智能手机市场放缓，很多山寨厂商不复存在，国产手机开始从低端往中高端爬坡，慢慢从混战中找到自我发展的道路。于是到了2016年，国产手机忽然崛起，积压的需求一下子爆发，去年简直成了国产手机的天堂。


在2016全球手机市场占有率及总销量排名中，国产手机中有三个品牌挤进了前五。华为、OPPO、Vivo在国内亦是三足鼎立，抢走了三星和苹果的市场份额。

不得不承认，在结束了苹果、三星霸占全球手机市场之后，华为、OPPO、VIVO等国产手机品牌的崛起让各界刮目相看，优异的品质、精准的产品定位使这些品牌们的市场占有率大幅提升，虽然还未能得以与苹果、三星等巨头比肩，但仍成绩斐然。

一时间，国产手机超越苹果、三星似乎已近在眼前，现在人们讨论的不再是追赶的问题，而是问：2017年，国产手机能否战胜苹果和三星？

2017年，会超过苹果三星吗？

一切看起来都顺理成章。三星Note 7的爆炸、苹果的创新乏力，国产手机品质的提升以及更贴合中国人的消费需求，所有这些似乎都预示着在智能手机市场，至少在中国，未来将是国产手机的天下。

如果一切都能按剧本书写，那是极好的，然而即便未来是光明的，但道路注定是曲折的。2017年一季度智能手机销量数据无疑给国产机泼了一次冷水。

2016年第四季~2017年第一季全球前六大智能手机品牌生产数量排名

榜单中最令人意外的是三星，本以为三星受到去年Note 7的爆炸事件影响，消费者信心减弱，可是虽然人家高端机表现不如预期，但中低端手机的表现仍可圈可点，凭借高性价比的J（Junior）系列一枝独秀，持续支撑销售动能，让三星成为第一季唯一季度正成长的手机品牌大厂。

再加上，三星最新发布的旗舰机种Galaxy S8系列颇受好评，全年来看，三星的销量将得到回升，国产手机超过三星又变得遥遥无期了。

既然超越三星不行，那苹果呢？苹果一季度市场份额大幅下跌，是不是意味着今年苹果手机表现不好？

其实不然，可以预期苹果一、二季度表现会不好，这是因为消费者在牟足了劲等待第三季苹果十周年发布的iPhone 8，今年必定是苹果的大年，iPhone 8很多大的改动早已不止让消费者们蠢蠢欲动，上游供应商们也是获益良多，为其加紧备货。

我们再看看，国产手机一季度的数据：

第一季度，中国品牌总生产数量季衰退达26%，其中华为第一季生产数量季衰退约22%。

OPPO、vivo，在2017年第一季的表现相较去年第四季，生产数量都有3成左右的衰退；小米今年第一季成长表现持平；而联想呢？早已消失在榜单之中，据TrendForce数据，联想在第一季度销量下滑近30%，与去年同期相比也下滑了近20%。

一季度国产机尤其是“华OV”的衰退又让很多人纳闷，今年这是怎么了？

其实一季度的衰退我们不必惊慌，一季度本就是淡季，再加上国内手机市场接近饱和，未来的竞争是存量市场的替换，所以自第二季度开始，各大厂商纷纷推出新款旗舰机，寄希望在iPhone推出前，获得一个好成绩，希望今年的销量能与苹果一战。

这就要回到上文了，在这种背景下，华为作为中国手机市场的老大哥被寄予厚望，希望在华为P10系列徕卡双镜头旗舰机种推动下，为国产手机品牌正名：中国手机将在中端往高端市场上继续高歌猛进，抢夺苹果和三星的市场份额。

可是，人算不如天算。从华为“闪存门”暴露出来的问题，不仅仅是存储器缺货那么简单，其背后是多年来国产手机的硬伤，也是未来几年真正制约国产手机发展的关键因子，这个问题就是：被抓住的命门！

被抓住的命门：缺芯少屏——有心杀“贼”，无力回天

犹记得3月27日，华为年度旗舰手机P10/P10 Plus登陆中国市场时，余承东豪言壮志：华为多项技术已领先苹果，华为手机在双摄技术、充电续航等方面已超越苹果。目前在中国高端手机市场能跟苹果竞争的只有华为。而距离发布会仅仅一个月不到，却被“闪存门”啪啪啪的打脸。

没错，中国手机需求巨大，中国人做手机可以在很多方面领先国外，双摄、充电技术、双卡双待等等，当然还有更为出名的互联网模式卖手机。这些在中国手机拓展中低端增量市场时发挥着巨大优势，然而当手机市场进入到中高端存量替换，关键零部件的更新换代成为未来趋势，这时比拼的是各大厂商手里的硬功夫：技术专利。

华为作为技术型企业，每年申请的手机专利排在世界前列，然而即便如此，华为在上游关键材料领域——存储芯片和屏幕，也是无可奈何。积贫积弱的中国半导体产业已经开始制约中国手机厂商的发展，试想，没有存储和屏幕的手机如何赶超苹果、三星？华为“闪存门”背后折射的是其有心杀“贼”，无力回天的悲哀。

被抓住的命门之一：存储芯片

几户任何一个电子产品都要用存储。2016年，全球半导体的总销售额为3400亿美元，其中Logic器件的收入占最大头，高达915亿美元；存储则紧跟其后，总营收为768亿美元。而在这个全球第二大的半导体组件里，中国几乎是一穷二白。

在主流的存储产品市场，三星、东芝、西部数据、美光、SK海力士和Intel几乎垄断了全球所有的NAND Flash市场；而在DRAM市场，则是三星、SK海力士、美光、南亚、华邦电子和力晶的天下。

从Dramexchange的数据显示，这几家厂商垄断了存储市场。也就是说按照2016年四季度的出货量统计，在NAND Flash领域，没有中国厂商的身影；而在DRAM领域，三星、SK海力士和美光更是占据了高达93.6%的市场份额，另外的供应商除了南亚、华邦电子和力晶以外，再没看到华商的身影。于是光在存储这个领域，这些供应商每年都从中国挣走了很多的外汇。

可怕的不仅是中国企业在存储供给市场上的缺失，还有我们超高的需求量和进口量。

2014年中国存储芯片市场规模达到2465.5亿元，占国内集成电路市场份额的23.7%。但存储芯片基本依靠进口，每年进口存储芯片的金额高达600亿美元。而无论是DRAM还是NAND Flash等领域，核心技术、专利等仍主要掌握在三星、东芝、SK海力士、美光、SanDisk、Intel等海外厂商手中，中国想要自己造存储芯片，可谓举步维艰。

而更为严重的情况是什么？对于中国手机厂商来说，是今年的价格疯涨+缺货。2016年Q4季度DRAM内存平均合约价格上涨了30%，今年Q1 DRAM也涨了15%；而NAND Flash也涨价超过20%，且涨价风暴将会持续到2017年底，绝大中国手机厂商本来毛利就不高，现在又面临给钱还买不到的局面，岂不悲哉。

被抓住的命门之二：OLED屏幕，国产手机面临3年空窗期

2017年是OLED发展极为重要的一年，随着今年iphone8将会使用OLED屏，无疑OLED的风口已经来了，未来一大批手机厂商使用OLED已成定局。而这块的霸主是谁？答案是三星。2016年其市场份额是96.7%。这也是为什么note7爆炸，但是三星2016年业绩诡异的同比递增10.7%的原因，就是16年屏幕这块利润暴增347%。在面板行业，三星目前是从上游机械设备、原材料、中游制造、下游面板全产业链通吃的。

而今年，最新的情况是：因为苹果订单的签约，三星AMOLED屏幕分配也出现新的变化，其中三星占据一半，苹果接近一半，剩下很少部分供应给中国几家手机厂商。

按照目前订单估计，如果苹果向三星抛去0.6-1.6亿片订单。2017年三星和苹果将消耗65%以上的AMOLED屏幕产能，如果三星扩冲产能至5.5亿片左右，留给其它厂商或不足2亿片。这也不难解释，从去年开始，一些国产手机，华为、蓝绿厂等旗舰机型纷纷缺货，以华为为例，当时市场传闻，三星推出Note7手机前，限制了对华为OLED屏幕的供应，导致华为Mate9系列手机推迟了上市时间。

苹果+三星对OLED的垄断，对中国手机厂商来说是釜底抽薪，那么，此时中国为何不加紧OLED的生产呢？

不是不想，实在是不能。要知道，目前三星的良率已经在90%以上，此时距离三星第一块AMOLED量产，已经过去十年，十年的沉淀换来90%的良率。而国内这块，首次出货超过100万片，以天马微电子、和辉光电 及昆山国显光电为主，只有40%-60%多的良率，由于良率达不到三星的水平因此成本显著高于三星。

其中，京东方是国内OLED的翘楚，据悉今年已经可以生产6代OLED屏幕了，不过，中国OLED厂商批量出货可能要等到2020年。换句话说，国产手机最重要的显示技术在这三年之中将被迫依赖韩系厂商！

国产手机会倒在进军中高端的路上吗？

我们经历了国产手机突进的2016年，但可能要面对艰难的2017、18年。进军中高端之路并不平坦，从苹果、三星手中获取市场份额本就是虎口夺食，再加上上游芯片、OLED屏幕供应链被他们死死抓住命门，前后夹击，国产手机会倒在这条突围之路上吗？

也许会，但我们依然前行。上游供应链突破不易：就上游存储芯片而言，紫光收购长江存储欲图打造世界顶级存储，中芯国际联手兆易创新和合肥市政府，也打造了合肥长鑫项目，国内芯片巨头正奋起直追，但要量产和追赶三星的工艺水平，实在太难。而在OLED领域，也要等京东方、华星光电和天马微电子等的良率爬坡以及产能形成规模了。

所幸在向中高端转型的路上，我们还有出路。上帝为你关闭了一扇窗，必定为你打开了另一扇窗。在国内手机产业逐渐饱和的情况下，数据显示，海外销售却日趋火爆。在印度，中国手机厂商的市场份额已经超过50%，在非洲已经超过40%，在欧洲市场也超过了20%。下图为印度智能手机市场的一些变化：

2016年第四季度，印度排名前五的手机品牌中，中国品牌就占了四个，分别为vivo、OPPO、联想、小米。市场份额方面，中国手机品牌占了46%，印度本土品牌萎缩到20%，其它品牌为34%。而华为呢？目前华为在全球有33个国家市场份额超过15%，另有18个国家市场份额超过20%，其中接近半数为欧洲国家，华为的国际化早已领先友商一大截。

当我们把华为黑的体无完肤时，很少有人考虑这背后的无奈；当我们把国产手机捧上天时，却不知危机已悄悄来临。

中国从手机的山寨国度到如今的在全球市场大放光彩，实属不易，然而还不到天下大定的时刻，居安思危，我们要把自己的命门给夺回来。
