他被“抓”去当将军，阴差阳错成为元帅，一手提携希特勒上台

六月 7, 2017 全球 本文总阅读量次

1914年对整个世界来说，都注定是不平凡的一年。在这一年里，帝国主义终于撕下那张虚与委蛇的面具，动起了真格，在利益与权益的支配下，战火首次烧向整个世界，这就是所谓的第一次世界大战。

这场世界性规格的战争影响了全球数亿人，也从此改变了他们的命运。这些人当中，有一个人或许在当时很不起眼，但毋庸置疑，这一年之后他的人生轨迹发生了改变。也因此，他能在日后改变世界。
他就是保罗·冯·兴登堡，德国政治家、军事家、魏玛（德意志）共和国第二任总统。

1914年之前，他本是一个赋闲在家的退伍军人。由于其在1903年做第四军军长时犯过一次致命失误，因此长期受到冷遇，也断了升迁之望，于1911年请求退役，回到汉诺威的庄园颐养。
这一年，他已年过花甲，六十四岁之龄的他在家没事养养花、种种草、喂个鸟、打个猎之类的，生活十分悠哉，谁都以为他的一辈子也就这样了，不会也不可能有大的变动。唯一有可能出现大的变动也只会是他突然暴毙。
不会有人想到，三年之后第一次世界大战会骤然降临。而他这个退役老将也在人手短缺之下被强行“抓”去再做将军，也因此他迎来了事业上的第二春。

战争爆发时，兴登堡正在波美拉尼亚他女婿的庄园里做客。听闻战争已经打响的消息后，他立即动身赶回汉诺威，毕竟波美拉尼亚地理位置十分微妙，处于德国与波兰之间，保不准就被战火所波及。

之后的几个月里，他曾幻想过再出江山的可能，但碍于年龄上有些偏高，这种可能性不是很大。但意外总是来的如此突然，1914年的8月，他收到一份来自总参谋部的任命书，派他去东线担任主将，与他搭档的总参谋长则是德国有名的手腕将军鲁登道夫。

兴登堡大概不会想到，他被任命是因为鲁登道夫向参谋部的点名请求，不然他这个老将可能根本没有机会在赴战场。而那位手腕将军看中的根本不是他的领导能力，而是他随和的性格与平庸的才能。

因为鲁登道夫是一个野心勃勃、锋芒逼人的人物，由于东线接连失利，他被迫从另一个战场调来作为东线的总参谋长。这样一来，他的搭档——也就是东线战场的主将则成了他发挥能力的羁绊。
他若想将他那一套战略计划顺利施展，与他搭档的主将自然要支持他，于是，他就向总参谋部点名兴求将，让随和的兴登堡跟他搭班子。

这一切似乎都在鲁登道夫的意料之中，上任之后的兴登堡果然对他听之任之，大小巨细皆交由鲁登道夫指挥。可以说他就是一个被架空的将军，实际上的指挥者是他背后的鲁登道夫。那么兴登堡每天都在忙些什么呢？关于这一点，德国名将霍夫曼给了我们一个解惑。

他曾在战地指着一个破旧的办公室，以挖苦的口吻对前来参观战地的人说：“这儿是战斗前元帅（即兴登堡）睡觉的地方，也是战斗后元帅睡觉的地方，准确来讲，这儿也是战斗时元帅睡觉的地方。”

兴登堡这样一位心大的将军后来被吹成“天才将领”，让真正知道内情的将领们十分不服，霍夫曼就是其中一位。他还曾写过书揭露兴登堡在第一次世界大战中的真实生活，但这并不能扭转毫不知情的德国民众对兴登堡的崇拜。

德国民众把在东线战场上赢得的战功都算在了兴登堡的头上，因此一战期间他的威望极盛。战争结束后，他也被晋升为元帅，享受着各种鲜花与掌声。
后来威廉二世下台，德国迎来重组，魏玛共和国在这样的形势下建立。

因为兴登堡在民众当中拥有极高的威望，在第二次选任时，他理所当然的被选为总统。而在他任职期间，不仅把希特勒给提了上来，还给希特勒扫平了很多障碍，为日后希特勒的独裁奠定了很好的政治基础。
就这样，一个本应在家种草养花的退伍军人，阴差阳错之下改变了世界。

如今的德国人已经看清楚这位将军的本性，于2013年镇全体投票并全票通过。撤销了兴登堡和希特勒两人在1933年颁予的名誉公民权。
