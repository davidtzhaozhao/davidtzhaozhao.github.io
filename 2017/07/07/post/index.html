“吃货”达·芬奇

七月 7, 2017 推荐 本文总阅读量次

　　达·芬奇喜欢画蛋。那个让我和小伙伴们记忆犹新的故事是这样的：达·芬奇把蛋畫了成百上千遍之后成了一个伟大的画家，画出了《蒙娜丽莎》《最后的晚餐》这样的杰作。

　　
　　
　　大人们总是喜欢以这样的故事提醒小孩：只要坚持不懈地画蛋，总会有收获的。

　　小孩当然不会听。实际上，大人和小孩都没有想过，达·芬奇为什么要画蛋，为什么别的画家都在画大自然，可达·芬奇偏偏要画蛋？有人查遍了达·芬奇的个人传记和其他资料，都找不到任何与画蛋有关的信息。不过文艺复兴时期的画家确实和鸡蛋分不开，他们不是画鸡蛋，而是把鸡蛋打碎，以蛋黄来调制颜料。这么说来，鸡蛋的确会在画家的画室里出现，这一点倒是毋庸置疑的。

　　绘本《世界上最大的蛋糕》讲的是达·芬奇的另一个故事。在这个故事里，达·芬奇是一个地地道道的“吃货”。吃着早饭时，他会想：“今天中午吃什么呢？”画着画时，他想：“画完画吃什么呢？”就连上厕所的时候都在想：“上完厕所吃什么呢？”总之，他的脑子里无时无刻不在想着吃。

　　和今天的许多“吃货”一样，达·芬奇还特别喜欢在厨房里捣鼓。他觉得捣蒜这件事太麻烦，要是有个捣蒜的工具就好了；拔鸭毛又特别费劲，要是有个拔鸭毛的工具就好了……总之，你有多想偷懒，就有多少好主意等着你。于是，达·芬奇制作出了很多工具，捣蒜器和拔鸭毛的工具很快就诞生了。他还制作了磨胡椒粉的工具、剁肉的工具、自动烧烤架、左撇子专用启瓶器、三股叉……他的厨房简直就是一个工具主义者的天堂。

　　你可能已经猜到了。绘本故事里的达·芬奇并没有规规矩矩地画蛋，而是做了世界上最大的蛋糕结婚礼堂。达·芬奇计算出要做好这个蛋糕礼堂需要17000块小蛋糕，然后像建造房屋一样，把蛋糕一块一块地搭建起来。为此，达·芬奇请来了建筑师、木工，他在料理笔记里这样写道：厨师应该学习建筑，因为如果不计算高度和重量，蛋糕就会倾斜或者整个儿倒塌。

　　后来我才知道，现实中的确有本著名的《达·芬奇笔记》。达·芬奇一生都在不断地写笔记，他的左右手都能写字作画，因此笔记里不少是用左手写成的反书——后人需拿镜子看才能破解。《达·芬奇笔记》里，藏着达·芬奇的各种好点子，充斥着“不可遏制的好奇心”和“极其活跃的想象力”。比如，画中的美人是源于模特的长相还是画家的手艺？猫在瓶瓶罐罐上跳舞会是怎样的景象？最后的晚餐是怎么吃的？大自然是否也给叶子安排了自己的法则？广场上什么是必不可少的？达·芬奇发明的飞机能飞起来吗？

　　达·芬奇没有上过名校，当然更没有多种多样的兴趣班可上，却成为画家、作家、雕塑家、发明家、哲学家、音乐家、医学家、数学家、生物学家、植物学家、解剖学家、地理学家、建筑工程师和军事工程师。和他同时代的法国国王法兰西斯一世对他的评价是：“世界上没有比他更伟大的人了！”

　　我们的父辈可能并不知道，达·芬奇小时候是个差生。他懒散，不爱上学，没有常性，对拉丁文更是恨之入骨。长大后的达·芬奇做事也经常半途而废，但他一辈子从没有停止寻找一种感觉，那就是对不可知的事物既害怕又渴望研究的兴致。

　　其实每一个孩子在面对不可知事物的时候，第一感觉都是害怕。我印象很深刻的是女儿学习游泳的时候，第一天很兴奋，第二天觉得很有进步，到第九天，突然用小手抠着泳池的边缘，不肯下水了。在孩子长大的漫长过程中，会有无数技能需要去掌握，最初的阶段都会是友好的，而在即将掌握到那个技能的时候，则会遇到一个黑暗期，只有平安过去了，才终会有所收获。而处于这个黑暗期的孩子，最需要大人们的支持。

　　生活中多半不会有学不会游泳的小孩、学不会骑车的小孩、学不会计算的小孩、学不会拼音的小孩，可大人们就是忍不住着急，然后把焦虑传递给孩子，以至于孩子的内心也有一种挫败感，觉得自己在这件事上就是“不行”。所以，算一算你陪孩子的时间，有多少是优质陪伴呢？你给予的方案，有多少保护了孩子的兴趣？今天的父母们一方面给孩子报各种兴趣班，期待孩子有旺盛的求知欲；一方面又忙着轻率地打断他，似乎越来越没有耐心去等待一个孩子自然的成长、简单的探索。

　　当不喜欢计算的达·芬奇需要做一个木工用具的时候，他就会很快学习相应的数学和物理知识，并马上成为一名卓越的学者。那么，还有什么能比有兴趣探索未知更重要呢？相对于考试、成绩、作业、证书，孩子在安静地做他喜欢做的事，比什么都重要。

　　《世界上最大的蛋糕》是一个比画蛋更有意义的故事。我看到一种轻松的人生，做着自己喜欢的事，让别人闭嘴吧。所以，今天的孩子可能更需要知道，达·芬奇不仅是伟大的画家，也是了不起的厨师，还是一个很棒的发明家。我的大学同学也发明过一样东西——能画椭圆的圆规。当年我们对他顶礼膜拜，现在来看，我相信是因为他在数学课上画椭圆画得烦透了。

　　如果觉得什么事情做起来不方便，去看看怎样才能更方便，这就是发明的开始。“吃货”达·芬奇告诉我们，经常做各种料理，可能会成为料理专家；在厨房里待得足够久去琢磨料理，又会发现料理和科学计算密不可分，然后会想成为科学家；当你为了科学去尝试做实验，又会发现颜色混在一起会发生变化，所以想成为画家。就是这样，当你沉浸于一件事情时，慢慢地就会对其他事情产生兴趣，只要饶有兴致地去探究，你就能做好很多事情了。
