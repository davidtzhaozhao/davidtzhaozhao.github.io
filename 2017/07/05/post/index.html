拿破仑vs希特勒巅峰战力对比，谁才是近代以来欧陆战争之王？

七月 5, 2017 全球 本文总阅读量次

忆往昔，一介炮兵少尉，奉命于危难之间，两线作战五次以少胜多击溃反法联盟，加冕法兰西第一帝国皇帝，颁布《拿破仑法典》.

成后世立法蓝本，穷途末路流落荒岛也能轻易颠覆复辟的波旁王朝，成法国历史上最受拥护和尊崇的民族英雄。

顾当年，区区鞋匠之子，受任于败军之际，挽经济危机之狂澜于既倒，扶战后德国之大夏于将倾，权至德意志第三帝国元首，兼并捷克、奥地利，干涉伊比利亚半岛，闪击波兰，击溃英法联军，与世界为敌，却被唾弃为邪恶恐怖的法西斯头子。

无论拿破仑还是希特勒，他们保家卫国的正义战争转化为侵略扩张的非正义战争之日，正是他们的帝国由盛转衰之时。但无论如何，他们曾经创下的军事奇迹都令人叹为观止，那么这两个传奇人物之间能够分出高低吗？

首先，对比的不是绝对战力，同一空间无论政治还是军事，从来长江后浪推前浪，战争规模不能同日而语，但通过对比对当时战争局势的整体把握和对关键战役的实际控制可以一窥两者差别。

法国大革命后，全欧惶恐，各国世传王公贵族担心牵一发而动全身，为了维护封建大地主的统治地位，组织了一次又一次反法联盟，然而从第一次反法联盟时期的土伦之战到第五次反法联盟时期的瓦格拉姆之战间近三十次大型战役拿破仑几乎一败难求，除了阿斯佩恩-艾斯林战役，但也无碍大局。

后来俄法战争使得法兰西帝国元气大伤，但击败拿破仑军队的不是俄国人民，因为拿破仑赢下了在俄罗斯的所有重要战役，却败给了天气和补给。

趁着法国远征俄罗斯失利，欧洲开始了第六次反法联盟，陷入困境的拿破仑依旧连战连捷、所向披靡，欧洲为之惊呼颤抖，他的谋略、他的果决、他的精准判断似乎开始重筑法兰西帝国的辉煌，却在关键时刻输掉了最重要的莱比锡会战，后来巴黎被趁虚而入，拿破仑被迫退位。然而这应该属于战争概率原因，只不过反法联盟输得起，拿破仑输不起。

流落厄尔巴岛的拿破仑仅以无上声望和权威轻易复辟，面对他的又是第七次反法联盟，拿破仑洞悉时势，坚决选择主动出击，凭借着惊世骇俗的军事智慧和能力打赢了林尼战役，却因为格鲁希的墨守成规输掉了本来胜券在握的滑铁卢之战，功败垂成，失去了重新统治欧洲的机会。

希特勒与拿破仑终究不同，与其说他军事奇才，其实更算是一个战争狂魔。或者说，比起军事家，他更像极端的政治家、野心家。他带着种族优越感和政党优越感开启了世界上规模最大也是最惨绝人寰的战争，仅一个多月与苏联瓜分波兰全境，又横扫丹麦、挪威，44天击败荷兰、比利时、卢森堡和法国，希特勒还不满足，兵锋直指英吉利海峡，他运用坦克闪击，飞机轰炸，潜艇奇袭的现代化作战手段，充分发挥了机械装甲部队的强大威力。

然而，有了一战的前车之鉴，希特勒仍然使德国军队陷入两线作战的困境，有了拿破仑的历史教训，希特勒还是纠缠于入侵英伦三岛、东征莫斯科和斯大林格勒，最后也走向了与一战德国和拿破仑同样的失败结局。

可以发现，从军事谋划、指挥作战和军心民心上，拿破仑更胜一筹，天要亡他，非战之罪。而希特勒在政治格局和对敌打击程度上优势明显，因为他眼中是全世界。
