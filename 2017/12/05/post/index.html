台湾十元硬币头像换成“便当阿嬷”吗？

十二月 5, 2017 两岸 本文总阅读量次

日前台湾网民提议，将十元硬币上的人头像改为长年以十元卖自助餐嘉惠穷人的庄朱玉女。网上联署超过5000人，获得中央银行回应。

按现行汇率，新台币10元相当于33美分、英镑25便士或2.19元人民币，庄朱玉女以这个价格出售便当，被称为“赔钱价”。

今年6月，有网民在公共政策网路参与平台提出“10元硬币头像，改铸10元便当阿嬷庄朱玉女”提案。

提案两天内就达到5000人附议的门槛，负责钱币发行的主管机关中央银行于9月27日出面回应。

网民提议的“便当阿嬷”名叫庄朱玉女。

“阿嬷”是闽南语中对于外婆、奶奶或年长女性的通称。

庄朱玉女西元1920年生于日本统治台湾时期的高雄州澎湖郡白沙庄吉贝。1936年与同村庄的庄文腾结婚，婚后冠夫姓，育有三子三女，并移居高雄市，与丈夫一同创业从事电器外线工程包办，后又从事码头货运装卸业务。

澎湖是台湾的离岛，与本岛最近的大城市高雄距离约150公里，从高雄出发的船班日间航行约四小时可到澎湖。澎湖以渔业、观光业为主，许多澎湖人选择到高雄工作。

庄朱玉女体恤澎湖同乡在高雄辛苦讨生活，1951年开始在位于高雄港码头的公司储料仓库给工人住宿，还免费提供便当，几年后因为花费太高，就改成可随意付费“吃到饱”的自助餐，随着物价上升逐渐从5毛、5元加到10元，从此不再加价，如果有生活困难获身无分文的人前来吃饭，则仍免费供餐。

庄朱玉女从购买食材到烹煮、结账都亲力亲为，菜市场的摊贩知道她在做善事，在菜价上也尽量提供优惠。她原本供应三餐，后来因年长身体渐衰才逐渐改为两餐、一餐，2001年获得高雄市政府颁赠“城市英雄”牌匾，但随后即因中风不得不收摊。

她的“赔本生意”经营了50年，为了持续“卖便当给穷苦人”的理念，庄朱玉女共卖掉七栋房屋。

2015年庄朱玉女过世，终年96岁。她的告别式有超过3000人到场，除了政界人士外，更多的是曾受她帮助的人。

高雄市民张华隆对BBC中文说，他30多年前吃过庄朱玉女卖的自助餐。

“我记得我有问阿姨‘可以挑几样菜？’阿姨说：‘弟弟你随便挑，挑完再算，’我就拿了一只卤鸡腿、一条香肠还有一片排骨，再加上高丽菜、白饭，白饭上面还浇了卤汁。”

“我爸还骂我拿太多了，我以为我这样子的组合应该是50元，结果结账是15元。”

张华隆补充说明，15元是他与父亲及同行码头工人三个人的价钱。当时的物价，一份鸡腿便当约是新台币30元。

“吃的人里面很多人都是一碗堆高高，有肉有菜的猛吃，当然白饭是无限量续碗的。我有观察，每个人吃完都会主动把碗筷收好然后交钱。其实阿姨只说收5元，但也是有人投10元或20元。”

当时的10元仍是纸钞。张华隆表示，用餐的人不会向庄朱玉女要回零钱。

他回忆：“吃饭的过程不会觉得这是人家施舍给我们的，我觉得就是一种被尊重的感觉。”

据他的观察，用餐的人很多，大部分是码头工人。有些人打完菜后还会回头多装一些，但是庄朱玉女不会制止，反而会一直劝这些工人多吃点。

张华隆说庄朱玉女为人低调，他是直到成年后看台湾的电视台采访，才知道“阿姨”的真名。

庄朱玉女过世后，儿子庄吉雄成立“庄朱玉女慈善会”，在澎湖发放低收入户慰问金和学生奖学金，租用巴士，供离岛学生上学。

庄吉雄在吉贝有经营旅馆，事业版图也跨足中国大陆。他於吉貝旅馆设立的庄朱玉女纪念园区2017年3月落成。当地政府官员出席剪彩。吉贝国小师生合力将“十元阿嬷”的故事制作成绘本，将阿嬷的故事流传下去。

根据国家发展委员会建置的“公共政策网路参与平台”规定，提案在60天内取得5000份附议成案后，主办机关就必须回应。

中央银行发文回应，称目前10元硬币发行量达38.5亿枚，六年前新增防伪功能、以国父孙文为头像的10元硬币发行，民众使用情况良好，市占量（32.4%）逐渐取代上世纪80年代的蒋介石版，目前没有再改版的急迫性。

央行表示，更换硬币头像即为改版，除了征集设计、通过设计、试铸、分配全国的程序外，自动贩卖机、缴费机等也都需要更新，“其程序繁杂，需时甚长，且耗费大量社会资源，绝非例行补充市面同版硬币所能比拟。”

央行建议社会先以其他方式纪念庄朱玉女，以她头像铸币将纳入下次改版的选项之一。

目前流通的10元硬币以蒋介石、孙文为头像的占最多数，共约35.9亿枚。另有六种其他图样，分别为“台湾光复50周年”、“新台币发行50周年”、“千禧年”、“中华民国建国90年”、“蒋经国百年诞辰”及“蒋渭水”，但较少见于世面。
