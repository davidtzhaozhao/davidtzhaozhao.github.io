不自由之港的一晚

八月 28, 2017 两岸 本文总阅读量次

七一前，自中国内地来港，一来想亲历回归二十年的历史现场，此外也想知道二十年後，香港到底有多不自由。

结果在短短的一日之内，亲历了香港的诸多不自由，令我如鲠在喉。号称五十年不变，才二十年就变得如此令人难以想像。

七一那天凌晨两点，我从半岛酒店去尖沙嘴海边散步，发现香港太空馆到文化中心全部被拒马包围，无法直接去到海岸。我从一个拒马缺口进去，遇到两个工作人员赶人，问他们为何如此，说是因为在维修。可是明明是半夜啊。这麽多年第一次遇到这麽大范围的拒马。不言自明，这不过是安保措施之一种而已。

今年七一大游行，政总广场大门紧闭，专门隔出警察通道，数万人就在此散了。添马政府总部跟过去的政府山相比，广场面积太小，我甚至怀疑政总在建筑设计上就是要杜绝示威的人群。而且，我认为，他们做到了。

晚上我与几位朋友在西营盘的一家天津饭馆吃饭。隔壁包间不时传来非常有中国体制内特色的掌声。我跟朋友说，肯定是中资企业或者中联办机关之类在这里吃饭，说不定就是庆功宴。後来朋友去偷偷听了一下，说有一个甚麽‌‌“局长‌‌”在里面。

几分钟後，一位胖胖的官员模样的人，约莫五十岁上下，被两位青年助手扶着走出房间，经过我们这桌时，我一位朋友突然对着他们大喊：‌‌“释放刘晓波！‌‌”官员模样的人并没有回头，但距离很近，显然是听见了。两位助手则回头看了一下我们这桌。过不久，饭店老板请我们这桌做东的朋友过去说话。

这位朋友是中国大陆来港的企业家，经常在此请客，与老板熟稔，而老板亦是大陆新移民。老板居然批评他：‌‌“你叫的甚麽人来这里？他有毛病吗？‌‌”这位朋友礼貌道歉之後，回到我们这桌气愤地说，以後再也不来这家吃饭了。我当时就说，即便是我们在上海的饭局上喊‌‌“释放刘晓波‌‌”，老板也未必会干涉，何况这是在香港。香港饭店也‌‌“莫谈国事‌‌”了吗？

香港是一个自由而多元的社会，客人在就餐时喊出有政治主张的口号，店家最多是认为不合适，而不是用一种不尊重的语气来教训客人。这不仅是不尊重客人，不专业，而且不尊重香港的多元与自由，是价值上的不尊重。一边要在香港挣大钱，一边还要鄙视香港的核心价值，何以分裂如此？

饭後，我们决定去中联办门口喊口号。到达之後，朋友们对着八楼亮灯的窗户大喊‌‌“释放刘晓波‌‌”。边上有一名警员很紧张，但并未制止。然後我们就发现中联办东侧拒马附近，有一名男子拿手机对着我们拍照。

这名男子身穿浅蓝色衬衣，头发较短。被我们发现後立即从我们身边走过去，走到中联办大门西侧的停车场入口那里站定。我们当时以为是路人，并未多加注意。随後，我们转到西边街，我一回头，发现拍照那名男子居然在跟踪我们。为了确认是跟踪，我们行至西边街和德辅道路口，站住不动。那位男子也停下来看手机。等我们到西区警署门口，他依然在跟踪，保持20米左右的距离。

我们立刻在警署的街角站定，也不走，然後一齐看着他。场面非常尴尬，他立刻过马路，假装去屈臣道与德辅道交叉口附近的巴士站等车。我们当时想要不要报警，这个人到底是谁？香港人还是大陆人？是甚麽身份？对我们拍照并且跟踪？

这个夜晚的故事并不恐怖，因为我们并没有受到甚麽具体的伤害。但是这样的故事可能发生在任何一个香港人身上。极权体制对待他们眼中的敌人，没有例外。时光倒转二十年或者更久，我不相信会在香港发生这样的事。

后来我们去中环七一吧喝酒，一位大陆朋友祝酒说，祝香港的今天比明天好。我们都沉默地哭泣了。
