阿根廷这个国家更惨

八月 17, 2018 全球 本文总阅读量次

一个曾经是南美最富的国家，一度被列入发达国家行列，沦落到今天的地步，简单是一部活生生的教科书。

如果说乌克兰是欧洲粮仓，那么阿根廷就是世界粮仓和肉类仓库，美国人均耕地面积多吧，阿根廷人耕地比美国人多两倍左右，还有丰富的石油，天然气，煤，铁，银，铀等等矿产，曾经是妥妥的拉美首富，人均收全球名列前茅。

现在通货膨胀率一路飚升，仅次于委内瑞拉，经济萎迷，物价飞涨，薪水冻结，苦不堪言。基本生活需要，除了食物之外，水，电，燃料等价格全部暴涨，上个月电费，煤气涨了三倍。

作为一个南美国家，阿根廷千好万好就一样不好，摊上美国这么一个邻居，说邻居其实也蛮远的，一个南美，一个北美，但美国把整个美洲视为后花园。—“美洲是美国人的美洲”。

阿根廷以前真可谓是：家里有屋又有田，生活乐无边。

二战之前就不说了，二战期间靠卖粮食和肉类发了笔横财，但它在政治站错了队，跟纳粹打得火热，所以二战后，政治地位比较低，联合国创始成员国就没它的份。

1946年2月，胡安·庇隆当选总统，阿根廷进入历史转折期，政治模式上，它是西方式的，经济制度上，它是社会主义式。阿根廷既不倒向美国，也不抱紧苏联，当时叫“第三条道路”，也叫庇隆主义。

庇隆和庇隆夫人（第二任妻子）的历史评说，不述，否则扯远了。

庇隆想干的事情就是要调整产业结构，由农业国向工业化进军，制定了五年计划。具体政策就是先搞国有化大企业，“一头沉”，轻农重工，想一口吃成一个胖子。

结果农业经济下来后，长远规划就困难重重了。对美国来说，它是绝不希望南美出现工业强国，所以庇隆政策就带有“反美”色彩。

但庇隆还是欢迎美资进入阿根廷，毕竟他不是一个布尔维什克，而是一个民粹主义浓厚的政治人物。

在对外关系上，正常的话，阿根廷跟新中国在1958年就应当进行建交谈判，结果被美国阻拦，加上阿根廷政局不稳，一直拖到了1972年。

1973年开始，南美形势大乱，智利总统阿连德9月11日被军队枪杀，皮诺切特成立了军政府，政变是CIA在背后一手操纵，另外巴西也在美国支持下成立了军政府，乌拉圭也发生了军事政变。

这一波军政府出现，原因是美国要镇压拉美风起云涌的左翼运动，如果不扶持军政府，不利用他们手里的枪炮，就无法屠杀左翼政治人物。

1976年，轮到了阿根廷，庇隆下台，军政府正式出现，据2007年解密文件，当时巴西和智利都跟阿根廷达成过协议，幕后总指挥在白宫，也就是基辛格。

美国新自由主义教主弗里德曼将阿根廷视为又一块实验田，他培养的阿根廷“芝加哥男孩”（经济学家）纷纷进入了阿根廷军政府之内，如财长，央行行长，国库司长等。

操刀私有化进程的是经济部长迪霍兹，他也是美国培养的，一上来就出售国营企业，解除资本管制，取消价格管制。

在短暂的繁荣后，资产流入了阿根廷利益集团手中，再转移到美国金融市场。

一年之后，工厂纷纷关闭，工资损失40%，在此之前，阿根廷的贫困率跟美国相差无几，失业率只有4.2%，这时，贫民激增，有的地区自来水停供，因为水厂私营，不赚钱就可以关闭，传染病开始四处蔓延。

老百姓最怕的是被关进集中营，左翼分子不是杀就是关，阿根廷军政府杀人还特别注重威慑力。

1977年，他们将一名左翼领袖吊绑在67.5米高的布宜诺斯艾利斯地标–白色方尖碑上，然后用机枪射击，震惊南美。

CIA还手把教他们如何使用酷刑，如何杀人，所谓失踪人口就是死亡人口，尸体哪去了？用直升机从空中抛进拉普拉塔河，还把死者胃切开，这样尸体就不会浮上来。

直到1983年民选政府重新出现，这七年到底杀了多少人？没有人知道。2006年9月，当时的首都警察局长艾切科拉兹被起诉，最终他被判处终身监禁。

这个判决具有历史意义，推翻了美国舆论所营造的是为了打击“恐怖分子”的谎言，整整一代左翼人士被杀戮，是一场有预谋的屠杀。

另外，阿根廷大学里的马克思主义经济学教授，都被称为“粉红色”经济学家，被关进监狱。

好笑的是，军政府要员想将从国企私有化捞到的不义之财转移到瑞士时，又不懂程序如何操作，只好偷偷去监狱里请教这些“粉红”教授，教授以此换取较好的监狱待遇。

这七年，军政府在美国经济学家的操纵下，彻底打断了阿根廷工业化进程，摧毁了一个庞大的中产阶级，十分健全的医疗，教育体系也被肢解。

社会上贫富两极，有钱的可以在佛罗里达，德克萨斯得到荣誉市民身份，没钱的只能在火坑里挣扎。人民唯一的集体娱乐就是足球场。

直到1988年，人们还在首都最豪华的“太平洋商场”地下室里发现了大批死难者生者刻写的名字和日期。

1982年与英国马岛战争，导致军政府解体。但美国已经牢牢控制了这个国家的经济命脉。

阿根廷毕竟农业基础强大，工业虽然单一，只要政策对路，政局平稳，不是不能恢复元气，事实上却陷入了一个死循环中。

1983年阿方索当选总统，这时阿根廷外债已由政变前的79亿美元升到450亿美元，它的债主有IMF，世界银行，美国进出口银行，华尔街私人银行。巴西，乌拉圭也一样，外债暴涨。

私有化打手，前经济部长迪霍兹以贪污和诈欺罪被逮捕。阿根廷当时新借的贷款中，有190亿美元流入私人手里，被转移到瑞士，成为就了阿根廷最富有的一群人，直到今天。

用国家名义贷款，私人分赃，酷刑和屠杀，这些事情完全被媒体过滤，今天很少有人知道阿根廷发生过什么，只知道委内瑞拉何等“独裁”。炒作谁？掩盖谁？美国媒体太懂了。

1990年阿根廷债务不减反增，到了650亿美元，总统阿方索赌了一把，发行大面额新货币，结果引发抢粮暴动，钱比纸本身还不值钱。

阿方索不得不提五个月辞职，重新大选，这样梅内姆在九十年代初登上了总统宝座人。被人民寄于厚望的梅内姆，却又任命了弗里德曼的信徒卡瓦罗出任经济部长，以向美国示好。

任命当天，阿根廷股市暴涨30%，卡瓦罗再带来一帮芝加哥经济学派的弟子，央行总裁弗南德兹，副总裁来自IMF博乌，总统经济顾问圭德提。

不单阿根廷，大多数南美国家都由美国培训的芝加哥学派弟子掌控经济政策，犹如兄弟会一般。

卡瓦罗在美国帮助下，控制住了比索暴跌局面，但他的私有化之路比迪霍兹走得更远。

四年时间内，卖掉了90%国营企业，如航空公司，石油公司，解雇了70多万名员工，买下这些企业的是花旗银行，波士顿银行，法国苏伊士公司，西班牙电信等等西方大公司。

这就是九十年代初，美国给全世界接受IMF贷款的国家带来的灵丹妙药–休克疗法。

国家又回到通货膨胀状态，但没有人敢反抗，大家都想，这总比军政府时期要好，等吧，阵痛过去就好了。

卡瓦罗这帮人给阿根廷带来了万劫不复的灾难，比索盯住美元，本土产品在价格上被国外进口产品打得满地找牙，失业率激增，金融上又受控于美国。

1998年亚洲金融风暴，带来巴西债务危机，为了避免南美爆发革命，美国让IMF再发贷款，拯救这些政权。

阿根廷陷入新债还老债，利息不能少的死循环之中，除了农产品之外，工业计划不要说实现，连制定都无法进行。

对阿根廷人民来说，他们认为是IMF这个吃人不吐骨头的恶魔夺走了他们的幸福，因为要偿还贷款，阿根廷政府必须按照IMF指令行事：消减福利，压缩公共开支，冻结薪水等等。

2000年宣布主权债务违约，2001到2006年，阿根廷农产品价格卖得比较好，咬着牙基本还清了上世纪老债务，宣布断绝与IMF的关系。

然而，新债务偿还并不顺利，因为阿根廷经济太依赖外国资本，到了2010年经济又开始恶化，外债越借越多，2017年底到了2330亿美元，美国是它的最大债权国，而中国是美国最大的债权国。

阿根廷想让中国拉它一把，克里斯蒂娜夫妇2003上台，老公当完总统，妻子当，两人算是中左派。

到2009年为止，阿根廷经济有所好转， 中国加大了对阿根廷农产品进口力度，委内瑞拉那时还挺富的，买了阿根廷国债。

奥巴马上来后，马上在后院拉拢各国，克里斯蒂娜出现了摇摆，对中国摆起谱来，阿根廷人也觉得自己国家比中国要强，民主嘛，自由嘛。那中国又何必出大力拉你一把？

今年，阿根廷外汇存款只有673亿美元，为了支付本息用掉了160亿美元，年底想要偿还的外债是650亿美元，必须再筹措490亿美元。

另外，250亿美元央行债券也即将到期，阿根廷现在是股票市场，债券市场，外汇市场全部受到重创。

5月份，又不得不向IMF借钱，借到了五百亿美元。比索一再贬值，投机资本开始逃离，为了留这些资本，阿根廷又提高了利率。

经济步步维艰，民生困顿不堪，政治乱成一锅粥。2015年超级富豪马克里当选总统，克里斯蒂娜拒绝参加权力移交仪式，一天之中阿根廷出现三位总统。

女总统说自己9月30日就离职了，而新总统接过权杖得等到10月1日中午12点。这半天没总统，由议长代总统。24小时内，一个国家居然有三位总统。

当然了，阿根廷人也习惯了，曾经有过两周五位总统的闹剧。

新总统马克里跟特朗普都在纽约搞房地产，两人私交不错，马克里当首都市长时，还将工程给了特朗普，让他大赚一笔。

结果特朗普当总统后，打贸易战连阿根廷也不放过，把马克里搞蒙了，指望兄弟拉一把，没想到特朗普还反踩一脚。

特朗普从阿根廷进口的生物燃料征收最高64％的关税，生物燃料产量阿根廷居世界第一，但90%出口给美国，现在特朗普王八拳打过来，阿根廷受不了。海军潜艇事故，也是祸不单行。

本来家里就没钱了，指望出口赚点，没戏了，阿根廷又去WTO喊冤。

中美贸易战一打，阿根廷有点小激动，中国不要美国大豆，这对阿根廷来说是大喜事，其它农产品也能跟上卖点。

还想提高中阿货币互换额度，向中国求助50亿美元。

对中国来说，马克里总统是右翼亲美政府，生意归生意，帮助就免谈了。阿根廷离希腊不远了，但希腊是欠德国钱，全民一公决就赖了，还嫌德国逼债过份。阿根廷主要欠美国钱，惹不起的。

阿根廷的悲剧只是南美国家一个缩影，不听话的，如委内瑞拉，玻利维亚被美国往死整，听话的，如阿根廷，巴西，也没落个好。

足球，不能当饭吃，虽然精神支柱，现在这成绩也夺走了阿根廷最后的快乐。

南美亲美国家道路显然出了问题，不是没路可走，首先得团结起来，共御强敌，再寻求体制变革。
