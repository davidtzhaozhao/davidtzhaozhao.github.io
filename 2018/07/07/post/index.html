王室该懂哪些礼数

七月 7, 2018 全球 本文总阅读量次

英国哈里王子与美国女演员梅根·马克结婚当天，女王在圣乔治大厅设午宴招待前去参加孙子婚礼的大约600名宾客。

晚上则由新郎的父亲，查尔斯王子，在距离温莎城堡一英里的浮若阁摩尔宫（Frogmore House）办喜酒晚宴，邀请200多位宾客。

在王室婚宴这样正式的场合，着装、举止和餐桌礼仪自有定规，见人打招呼或相互引见时也须注意使用正确的爵位勋衔，公、侯、伯、子、男爵，不能搞错。

以哈里的哥哥、嫂嫂，也就是剑桥公爵夫妇当年的婚宴为例，中午的自助餐可以随意，但正式晚宴就很讲究了。如果哪天应邀去这种正式场合，提前熟悉相关礼仪总没坏处。

进入大厅，找到自己的餐桌和位子，入座。眼前是摆放得整整齐齐、错落有致的各种餐具。

刀、叉、匙
很简单，根据就餐进程，从外向里依次拿。如果菜单上第一道是汤，那汤匙总是放在最右边，如果是第二道，就是从右边数第二把。甜食餐具永远摆在餐盘上方，匙上叉下，匙向左，叉向右。

杯子
杯子的摆放也是按使用的先后顺序。比如，水杯、香槟酒杯、白葡萄酒杯、红葡萄酒杯、甜食酒杯。

餐巾可以放在左手边，或者叠放在餐盘上。

姿势
进餐方式视情况而定。有些食物比较特别。

餐前面包：不能用刀切开，要用手掰开。
汤：将汤碗/杯略倾，用汤匙由向外舀；将汤送入嘴里；不能舀太满，要小口抿，不可大声吸进嘴里。
芦笋：直接用手指捏起来吃，从芦笋头开始。
蚝：用蚝叉将蚝肉与壳分离。用拇指、食指和中指拿住蚝壳，举起，置于下唇，微倾，让蚝肉和汁水滑入口中。不要整个囫囵吞下，得细嚼品味。

哈里和梅根的婚礼请柬上写着着装要求，规定男士需着”礼服或西装”，女士需着”礼服搭配帽子”。

这意味着男士需要衬衫、西装、领带，并可以搭配礼帽及马甲，而女士则需注意裙装过膝，且帽子的选择需与参加皇家赛马会时相似。

当年威廉王子和凯特的婚礼，请柬上对宾客的着装要求写着制服、礼服和鸡尾酒会礼服。

在英国，只有现役军人可以穿制服参加婚礼，而且需要得到上级军官批准。

威廉王子婚礼当天穿的是爱尔兰卫队骑兵军官制服，哈里王子当伴郎，穿的是皇家近卫骑兵制服。

男宾通常要穿灰色或蓝色正式礼服，灰色或黄色背心，配灰色条纹长裤，再加一顶灰色礼帽，在室外戴，进室内用胳膊夹着。领带是必须的，或者有别针的领结；还要穿单色衬衫配袖扣。

拍正式合影时不能戴礼帽。

手套要跟背心搭配，不戴时要拿在手里。脚上是黑色系带皮鞋。

女士礼服可以是有外套的裙装加礼帽和手套，高跟鞋，搭配手提包。

鸡尾酒会礼服，对男士而言就是西装领带，对女士而言是指正式套装，礼帽自便。

可以想象，戴着漂亮的宽沿礼帽的女士要行贴面礼时，会比较麻烦。

知道一点基本的礼仪应该有助于克服这个难题。

屈膝礼
屈膝礼属于正式礼仪。在婚礼场合，一般来说原则上是不要做得太过分。

根据现代礼仪手册， 那种大幅度的屈膝和弯腰深鞠躬已经过时了，在王室圈子里要是谁行这样正式的大礼，很可能被人笑话。

礼仪专家认为，在见到女王时，女士可以象征性地浅浅屈膝，男士则可以点一下头，注视地面片刻；道别也一样。

禁忌
跟女王握手可以，但其它的肢体接触就不行。前澳大利亚总理基廷1992年出了国际大洋相，被人戏称为“澳洲的蜥蜴”。为什么？就是因为他把手搭在女王背上，被人拍了照片。

不过，2015年，女王本人跟美国前第一夫人米歇尔·奥巴马相互搂着肩背，也有像片为证。当时女王在白金汉宫设宴招待访英的奥巴马总统夫妇。

白金汉宫发言人后来解释说，那是因为女王和奥巴马夫人相互欣赏、相互都很喜欢对方。

需要注明的是，较年轻的王室成员，比如威廉王子和哈里王子，对这些见面礼仪很可能不太叫真。

另一个禁忌是拍照摄像。白金汉宫发言人说，不光是王室婚礼，任何王室活动场合都不鼓励来宾用自己的相机拍照摄影。

按照惯例，王室活动都有指定的专业摄影师雨果·伯南德（Hugo Burnand）摄影。

通常有爵位的贵族们不需要穿相应的勋爵礼服，但像议会每年开幕之类的正式典礼，就必须这样礼袍盛装出席。
爵位勋衔
哈里和梅根的婚礼请了不少平头百姓参加。不过，王室大婚，出席婚庆喜宴和酒会的少不了皇亲贵戚，见面称呼时要记得使用正确的爵位头衔。

王室成员以下，是世袭贵族：”公、侯、伯、子、男“五爵。现在已经很少再册封新的世袭贵族。

王室婚礼通常不需穿礼袍。

公爵/公爵夫人或女公爵（Duke / Duchess）
公爵是五个勋衔里最高的爵位（公、侯、伯、子、男），在英国一般只有王室宗亲才授公爵勋衔。如果公爵还是现役军人、在教会担任圣职，或者出任驻外大使，则他们的称谓顺序是先官职后勋衔。向人介绍时，公爵的夫人就叫公爵夫人，或者某某公爵夫人 。

今日英国还有大约20位非王室成员公爵在世。在正式场合，公爵穿深红色的丝绒外套，帽子上镶四条貂皮，冠冕上有一个金环，上饰八枚金叶片。

侯爵/侯爵夫人或女侯爵（Marquess / Marchioness）
公爵下面是侯爵/女侯爵或侯爵夫人。在被引见时，对他们的正确称呼是勋爵 （Lord）或夫人（Lady）。现在有大约34位在世侯爵。

由于历史原因，英国获封侯爵的数量是五等爵位里最少的。在正式场合，侯爵穿红色丝绒外套，帽子上有三条貂皮，冠冕上镶一个银环， 上饰四枚金叶和四个银球。

每年女王宣布议会开幕，是贵族和议员们必须盛装出席的正式典礼。
伯爵/伯爵夫人或女伯爵 （Earl / Count or Countess）
世袭勋爵里第三级是伯爵或女伯爵。这是英国历史上出现最早的爵位，始于10世纪。现时在世的伯爵、女伯爵有大约190多位。

在重大正式场合，伯爵穿镶白色毛皮边的红色丝绒外套，软帽上镶三条貂皮，冠冕上有一镀金银圈，上沿饰有八个银球。

子爵/子爵夫人或女子爵（Viscount / Viscountess）
第四级勋衔，子爵，源于法国，最早是伯爵的副手，也是从10世纪开始就有。后来也有一方诸侯获子爵衔。英国子爵勋衔始于1440年，”百年战争“期间， 亨利六世当上英格兰和法兰西国王后整合两国的勋衔。

正式场合子爵的帽子上镶有两条半貂皮，冠冕上加一银环，饰六个银球。

称呼子爵时比较简单，就是某某勋爵（Lord X），而不必称某某子爵（Viscount X）。正式爵位只用于法律文件或正式仪式典礼上。

男爵/男爵夫人或女男爵（Baron / Baroness）
这是五个勋爵里最低一级爵位。在古法语里这个词的意思是”自由人“。

英国最早的男爵是1387年英王理查二世授予的基德敏斯特男爵（Baron Kidderminster）。

通常称呼男爵跟称呼子爵一样，用通称Lord/Lady，即勋爵/夫人而不必说具体爵位。正式爵位只用于法律文件或正式仪式典礼上。

苏格兰的世袭贵族勋衔称谓跟英格兰一样。

除了五个传统世袭贵族，非世袭终身贵族也可被册封为男爵/女男爵。非世袭终身贵族只授予本人，不传代。英国每年两次册封，一次是君主生日，一次是新年。

册封贵族的目的是表彰在某个领域有突出贡献的人士，或者是为了使某些人士进入上议院。
